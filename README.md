# Power-BI-Project
Adidas US sales and profit performance Analysis

This project features an interactive Power BI dashboard built to uncover sales trends over time, evaluate product- and city-level profitability, and assess how different sales methods impact profit performance.

## Key Insights
- Identified sales and profit trends over time  
- Highlighted top- and low-performing regions  
- Analyzed factors impacting overall profitability  

## Tools Used
- Power BI  
- DAX  
- Power Query
